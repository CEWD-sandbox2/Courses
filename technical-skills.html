Computer Essentials

Course aim

This course provides an introduction to skills in using a SLHD Personal Computer (PC).

Course objectives
  • Become familiar with computer terms.
  • Start and log in to a SLHD PC.
  • Identify computer equipment and features.
  • Access the SLHD Intranet to locate commonly needed links.
  • Create and save a word file.
  • Access and modify an Excel spreadsheet.
  • Access Outlook account and view features.
  • Identify sources of Help when online.
Target group
  • LHD staff that have little or no experience working with computers

Microsoft Excel Intermediate: Formulas, Functions and Lists

Course aim

This short workshop is for staff using MS Excel 2013 regularly in their role, needing an enhanced ability with lists, functions, formulas. Participants should already be competent in basic Microsoft Excel skills.

Course objectives

By the end of this course participants should be able to use the following functions:

Formula calculations

  1. The formula bar, cell content vs cell value
  2. Fill down, when to use relative and absolute references
  3. Precedents and dependents
  4. Formula auditing
  5. Error checking
  6. Text to columns
  7. What if analysis.

Functions

  1. Autosum (), sum all rows and columns
  2. Arguments - form and multiple arguments
  3. Vlookup()
  4. Create names from list
  5. Using names in functions
  6. Text functions: Len, Left, Right, Mid, Find, Trim
  7. Indirect - lookup a lookup

List functions

  1. Design, auto-complete, using pick lists, applying data validation, filling in gaps in data
  2. Go to special
  3. Freeze frames

Named areas

Target group
  • Staff using MS Excel 2013 in their role
  • Staff needing an enhanced ability with lists, functions, formulas

Microsoft Outlook and MS Teams

Course aim

This short course provides the participant with knowledge and skills to more effectively manage their communications and workload using the features of Microsoft Outlook and MS Teams in the windows environment. Participants should already be competent in using a mouse and keyboard, and have an understanding of the Windows environment. Participants need to have an email and MS Team account set up by ICT prior to attending.

Course objectives
  • Identify key tools in MS Outlook 2019
  • Create a meeting and invite participants
  • Create tasks, notes and folders
  • Identify ways to organise emails
  • List effective email tips
  • Identify features of MS Teams: chat, teams, calls and files
  • Demonstrate communications by text, call or videoconferencing MS teams
  • Share and collaborate on files in MS Teams

My Health Learning - Operational Assessor Role - virtual (Zoom)

Course aim

This session will provide a demonstration of the OA (Operational Assessor) Role in My Health Learning. This role provides the ability to enter candidates as “competent” or “not yet competent” in skills assessments (OAs or Operational Assessments) in My Health Learning. This role also allows you to run a report on the status of candidates and whether or not they have been deemed competent in a particular skill.

This session also covers the OA Data Entry role.

Course objectives
  • Understand the role of the OA Assessor
  • Be able to record the outcome of a competency assessment in My Health Learning
  • Be able to generate a report on staff who have been deemed competent or not yet competent in a competency assessment
Target group
  • Staff within SLHD who have been allocated the OA Assessor role in My Health Learning

My Health Learning - Scheduler Role - virtual (Zoom)

Course aim

This session will provide a demonstration of the Scheduler Role in My Health Learning. This role allows you to manage classrooms in My Health Learning. It will allow you to create, duplicate, postpone or cancel classrooms (scheduled offerings), enrol participants directly into classes, and generate attendance sheets and record attendance at training sessions. It also allows you to run reports on workshop completion numbers, and generate a transcript of a student’s learning history.

Course objectives
  • Understand the role of the Scheduler in My Health Learning.
  • Be able to navigate the dashboard of the scheduler home page.
  • Be able to describe classroom objects and scheduled offerings.
  • Be able to create and duplicate scheduled offerings
  • Be able to schedule multiple sessions of the same course
  • Be able to differentiate between published and unpublished classrooms
  • Be able to enrol participants in a scheduled offering and record attendance
  • Be able to generate attendance sheets.
  • Be able to run reports on classroom completions
  • Be able to generate a transcript for an individual
  • Be able to manage staff on extended leave or who have resigned
  • Be able to manage incomplete offerings
Target group
  • Staff within SLHD who have been allocated the Scheduler role in My Health Learning
View and apply on My Health Learning
